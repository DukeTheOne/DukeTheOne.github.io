## 1.3 - Logical Equivalence

> Two statements P and Q are logically equivalent if they have identical truth values in every row of a truth table.

### Fundamental Logical Equivalences

```
Idempotent:      Commutative:         Associative:                     Distributive:
p ∨ p ≡ p        p ∨ q ≡ q ∨ p       (p ∨ q) ∨ r ≡ p ∨ (q ∨ r)        p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r)
p ∧ p ≡ p        p ∧ q ≡ q ∧ p       (p ∧ q) ∧ r ≡ p ∧ (q ∧ r)        p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)

Identity:        Domination:          Negation:                        Double negation:
p ∧ T ≡ p        p ∨ T ≡ T            p ∨ ¬p ≡ T                       ¬(¬p) ≡ p
p ∨ F ≡ p        p ∧ F ≡ F            p ∧ ¬p ≡ F

Absorption Laws:                      De Morgan’s Laws:                Implication Equivalences:
p ∨ (p ∧ q) ≡ p                       ¬(p ∧ q) ≡ ¬p ∨ ¬q               p → q ≡ ¬p ∨ q
p ∧ (p ∨ q) ≡ p                       ¬(p ∨ q) ≡ ¬p ∧ ¬q               ¬(p → q) ≡ p ∧ ¬q
                                                                        p ↔ q ≡ (p → q) ∧ (q → p)
Contrapositive:
p → q ≡ ¬q → ¬p
```

### Simplify with Equivalences

For example, (p → q) ∧ p \
≡ (¬p ∨ q) ∧ p \
≡ (p ∧ ¬p) ∨ (p ∧ q) \
≡ F ∨ (p ∧ q) \
≡ p ∧ q

### Tautology vs Contradiction vs Contingency

A **tautology** is always true. \
A **contradiction** is always false. \
And a **contingency** can be either true or false depending on the situation.
